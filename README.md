# Snake Xenia - Two Player Snake Game

A competitive two-player snake game with time-based gameplay and unique scoring mechanics.

## 🕹️ Controls

### Player 1 (Blue Snake)

- W: Move Up
- S: Move Down
- A: Move Left
- D: Move Right

### Player 2 (Green Snake)

- ↑: Move Up
- ↓: Move Down
- ←: Move Left
- →: Move Right
